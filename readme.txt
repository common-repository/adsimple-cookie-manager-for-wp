== Description ==

Mit diesem Plugin können Sie in nur drei Schritten den AdSimple® Consent Manager auf Ihrer Website einbinden.

1. Plugin auf Ihrer WordPress Seite installieren und aktivieren.
2. AdSimple ID aus dem Consent-Manager auf adsimple.at in das Eingabefeld kopieren.
3. ID SPEICHERN klicken.
4. Alle weiteren Einstellungen nehmen Sie im Consent-Manager auf adsimple.at vor.
5. Wenn Sie eine geordnete Liste aller Cookies auf einer Unterseite anzeigen möchten, dann können Sie das durch Einfügen des Shortcodes [acm_cookie_list] ganz einfach umsetzen.


Was genau der AdSimple® Consent Manager alles kann, können Sie unter dem nachfolgenden Link ansehen:
https://www.adsimple.at/consent-manager/


== Installation ==

1. Plugin auf Ihrer WordPress Seite installieren und aktivieren.
2. AdSimple ID aus dem Consent-Manager auf adsimple.at in das Eingabefeld kopieren.
3. ID SPEICHERN klicken.
4. Alle weiteren Einstellungen nehmen Sie im Consent-Manager auf adsimple.at vor.

== Frequently Asked Questions ==

= Ist der AdSimple® Consent Manager kostenlos? =

Ja. Der AdSimple® Consent Manager ist ein einer kostenlosen Variante verfügbar.

= Ist eine Registrierung auf adsimple.at notwendig? =

Ja. Über Ihr Benutzerkonto können Sie mit dem AdSimple® Consent Manager einen oder auch mehrere Cookie Hinweis Scripte zentral verwalten.

= Was ist die AdSimple® ID? =

In Ihrem Dashboard auf adsimple.at unter dem Menüpunkt "Consent Manager" finden Sie die notwendige AdSimple® ID um die Verbindung zu Ihrer Website herzustellen und damit den Script-Code einzubinden.
